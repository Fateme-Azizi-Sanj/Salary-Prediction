# Salary Prediction

This project predicts employee salaries based on years of experience.  
The dataset is stored in salary_data.csv, and the model is implemented using Python and scikit-learn.

## How to Run
1. Install dependencies:
   `bash
   pip install pandas matplotlib scikit-learn

2. Open the notebook:

jupyter notebook notebooks/salary_prediction.ipynb


Description

Load and explore the dataset

Split data into training and test sets

Train a simple regression model

Predict salaries and visualize results
